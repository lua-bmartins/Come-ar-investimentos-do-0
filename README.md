# Come-ar-investimentos-do-0
Este projeto consiste na criação de um podcast curto gerado com Inteligência Artificial, com o objetivo de demonstrar como as IAs generativas podem ser utilizadas para produzir conteúdos realistas, naturais e com qualidade profissional.
Todo o processo envolveu roteirização com apoio de IA, refinamento de linguagem para soar o mais humano possível, e geração de voz por IA para a narração, explorando conceitos como Prompt Engineering e modelos de linguagem.

O projeto faz parte do Lab Natty or Not da DIO e tem como principal objetivo investigar até que ponto conteúdos criados por IA podem se aproximar da naturalidade e da comunicação humana, servindo também como peça de portfólio para demonstrar aplicação prática de IA generativa em mídia e comunicação digital.
